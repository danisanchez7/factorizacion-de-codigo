## Configuración en MacOS y Linux
Ejecute los siguientes comandos en el terminal:

python3 -m venv .venv && 
source .venv/bin/activate &&
source setup.sh

## Configuración en Windows
Ejecute los siguientes comandos en el terminal:

python3 -m venv .venv &&
.venv\Scripts\activate &&
setup

## Ejecute el siguiente comando en el terminal:
